Assignment 5

This Python script implements a K-Nearest Neighbors (KNN) classifier to predict customer churn in the banking industry. It addresses the assignment requirements by developing a custom KNN algorithm from scratch, without using pre-built libraries for KNN. The code preprocesses the provided banking dataset, handling both numerical and categorical features. It then performs hyperparameter tuning to find the optimal value of k, using cross-validation to evaluate model performance based on ROC AUC scores. The script visualizes the performance of different k values, trains the final model with the best k, and generates predictions for the test set. Finally, it creates a submission file with the predicted probabilities of customer churn, fulfilling the assignment's goal of building a predictive model for customer retention in the banking sector.]

Midterm

This project aims to predict Amazon movie review ratings using a machine learning pipeline that analyzes review text and metadata. Data preprocessing includes cleaning text data and handling missing values, while feature engineering extracts sentiment scores, text-based metrics (like review length and punctuation), and user/product averages. Logistic regression is chosen for its interpretability, with optimal hyperparameters selected through grid search and cross-validation to maximize predictive accuracy. The code integrates pandas for data handling, NLTK for sentiment analysis, and scikit-learn for model training and evaluation, using TFIDF vectorization to capture text information and generate predictions.
